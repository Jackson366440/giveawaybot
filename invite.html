GiveawayBot V2 Setup

This page will guide you through the setup of the bot. Because the bot is still in its experimental stages, a server has not been provided to host the bot. The bot will have to be hosted locally.
To simplify the process of the setup required to self host the bot, a setup wizard has been provided to automatically configure and run the bot.

Invitation

Invitation

You will need to create a bot and grant it the required permissions before inviting it. Luckily, a bot and invitation with everything required is provided. Simply invite the bot via the link above.

Setup

Setup Wizard Version 2.0.1 Experimental

Once the bot has been invited, configuration is necessary. To start configuration, simply download and run the setup wizard above. The setup wizard will automatically adjust settings and
install any required frameworks/prerequisites. After setup, the program will automatically close.

Important Notes

Please note that since the bot is self hosted, the bot will have to be configured properly before usage. If you find that the bot is not working, first try running the setup wizard again. If it still
does not work, it is most likely because the bot is down for maintenance. As the bot is still in development, downtime from maintenance will happen occasionally.
